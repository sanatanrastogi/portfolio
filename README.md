Sanatan Rastogi — Personal Portfolio
A simple, responsive single-page portfolio website built with HTML, Bootstrap 4 and minimal JavaScript. It presents education, technical skills, certifications, experience, projects and contact information.
Features
•	Clean, mobile-first layout using Bootstrap 4
•	Sections: Title, Education, Skills, Certifications, Experience, Work, About, Contact
•	Animated Education section using Animate.css and jQuery
•	Image zoom effect for certificates
•	Live clock in the footer
•	External links to resume, certificates and social profiles
Files
•	webportfolio.html — main site page
•	Images and PDFs (e.g. C.jpg, C.pdf, SanatanRastogi.pdf) — media assets used by the page
Prerequisites
•	Modern web browser (Chrome, Edge, Firefox)
•	(Optional) Static server for local testing (recommended for some browser behaviors)
Run locally
1.	Open directly:
•	Double-click webportfolio.html or open it in your browser.
2.	Serve with a simple HTTP server (recommended):
•	Python 3: python -m http.server 8000 then visit http://localhost:8000/webportfolio.html
•	VS Code: Install and use the Live Server extension
Notes & troubleshooting
•	If commit or file operations fail with "file is being used by another process", close any editors, preview panes or browsers that may have the file open, or restart your system. Use Resource Monitor / Process Explorer to find locking processes.
•	Ensure external CDN resources (Bootstrap, jQuery, Font Awesome, Animate.css) are reachable; consider bundling assets for offline use.
